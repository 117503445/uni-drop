# uni-drop

frontend of [UniDrop](https://www.unidrop.top)

## technology stack

- React
- React Router
- Redux Toolkit

- Vite
- Vitest

- TypeScript
- Tailwind CSS
- Playwright

- PeerJS

- ESLint
- Prettier

## operations

```sh
# run
pnpm run dev

# build
pnpm run build

# test
pnpm run test

# check (ESLint + prettier + build)
pnpm run check

# generate fake names to ./src/utils/fake-names.json
pnpm run gen-fake-names
```
